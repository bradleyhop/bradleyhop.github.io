JavaScript Calculator for freeCodeCamp

Texas Instruments SR-10

This calculator fulfills the "Front End Libraries Projects - Build a JavaScript Calculator" project for freeCodeCamp. The design is inspired by one of the first scientific calculators, the Texas Instruments SR-10. The calculator originally sold for $150.

This app is built using Vue and Sass. The math.js library handles the calculations and arithmetic operations using formula/expression logic. There are 7 places of precision chosen to reflect the SR-10's display limitations.
